# 跑者廣場訓練手冊生成器 — 設計規格

**日期**：2026-06-22  
**狀態**：已確認，待實作

---

## 一、一句話摘要

`site/trainer.html`：單頁跑者訓練計畫生成器。用戶填 9 個欄位 → 生成含熱身/主課/收操的完整週曆菜單 → PDF 或手機 HTML 匯出。資料全存 localStorage，不上 server。

---

## 二、架構

```
site/trainer.html          ← 唯一新增檔案，不動現有 index.html
  ├── View: Setup          ← 輸入表單
  ├── View: Plan           ← 計畫視圖（進度條 + 週曆）
  ├── View: Log            ← 訓練紀錄
  └── View: Check-in       ← 週評估（嵌在 Plan 視圖底部）

localStorage key: "runner-trainer-v1"
  └── { profile, plan[], log[], checkins[] }
```

從 `site/index.html` 導覽列加入「訓練計畫」連結指向 `trainer.html`。

## 三、輸入表單（Setup View）

### 欄位清單

| # | 欄位 | 類型 | 說明 |
|---|------|------|------|
| 1 | 訓練目標 | 卡片選擇 | 5K/10K｜半馬 21K｜全馬 42K｜傷後重建 |
| 2 | 目標日期 | date | 自動算訓練週數；<8週警告，>24週提示 |
| 3 | 目標完賽時間 | text (H:MM:SS) | 推算目標配速、節奏跑配速、間歇配速 |
| 4 | 訓練日（星期幾） | 圓形按鈕多選 | 點一次=訓練日，再點=長跑日（深藍），再點=取消 |
| 5 | 長跑日最長可跑時間 | select | 60/90/120/150+ 分鐘 → 限制長跑距離上限 |
| 6 | 目前每週跑量 | number (km) | 計畫起點；填 0 表示剛開始 |
| 7 | 輕鬆跑配速 | text (M:SS) | 推算目前能力；與目標時間做差距評估 |
| 8 | 最近比賽成績 | text，選填 | 有值時用 VDOT 表校正訓練配速 |
| 9 | 身體狀況 | 多選 pills | 無傷｜腳踝｜膝蓋｜足底筋膜炎｜其他 |

### 即時計算區（表單底部）

填入目標日期 + 目標時間 + 配速後，即時顯示：
- 訓練週數
- 目標配速（/km）
- 節奏跑配速（目標配速 +12秒）
- 間歇跑配速（目標配速 -10秒）
- 訓練天數/週
- 目標難度評估（目前推估完賽 vs 目標完賽時間差距）

## 四、計畫生成邏輯

### 階段劃分

| 目標 | 階段結構 | 最短週數 |
|------|---------|---------|
| 5K / 10K | 基礎 → 強化 → 減量 | 8 週 |
| 半馬 | 基礎 → 強化 → 高峰 → 減量 | 12 週 |
| 全馬 | 基礎 → 強化① → 強化② → 高峰 → 減量 | 16 週 |
| 傷後重建 | 輕量 → 漸進 → 鞏固 → 維持 | 8 週 |

### 規則

- **減量週**：每 4 週自動插入，跑量 -20%，移除間歇，保留長跑（縮短）
- **賽前減量（Taper）**：最後 2–3 週跑量 70% → 50% → 20%
- **跑量增長**：每週 +10%（跳過減量週）；起始跑量 = 目前跑量 × 1.05
- **有傷時**：移除間歇跑，降低節奏跑比例，加強熱身/收操說明

### 每週日課表分配（依訓練天數）

| 天數/週 | 課表結構 |
|---------|---------|
| 2 | 長跑 + 輕鬆跑 |
| 3 | 長跑 + 節奏跑 + 輕鬆跑 |
| 4 | 長跑 + 間歇跑 + 節奏跑 + 輕鬆跑 |
| 5 | 長跑 + 間歇跑 + 節奏跑 + 輕鬆跑 × 2 |

長跑固定排在用戶指定的「長跑日」，其餘依用戶選的訓練日順序分配。

### 每日課表結構

```
熱身（5–15 分鐘）
  走路 / 動態伸展 / 慢跑暖身
主課（距離 or 時間 + 配速目標）
  輕鬆跑｜節奏跑｜間歇（組數×距離 + 休息）｜長跑
收操（5–10 分鐘）
  慢跑降速 + 靜態伸展
[選配] 肌力/核心（10 分鐘）
  傷後重建：每訓練日必附；一般計畫：間歇日 + 長跑日附加
```

### 休息日

休息日卡片顯示選配恢復建議（根據傷況自動建議，可手動選）：
- 完全休息
- 主動恢復（散步 + 泡沫滾筒）
- 伸展 / 瑜珈（20 分鐘）
- 交叉訓練（游泳 / 騎車，不超過 60 分鐘）

## 五、計畫視圖（Plan View）

### 佈局

```
[導覽列] 本週課表 | 週評估 | 訓練紀錄 | ⚙️修改設定

[進度卡]
  半馬訓練計畫 · 目標 2026-10-25 · 目標配速 5:41/km
  第 3 週 / 共 16 週 ████░░░░░░░░ 19%
  完成 2 次 | 本週 16/28 km | 累積 54 km | 遵從率 71%

[階段分頁]
  ✅基礎建量 W1-4 | 🔥強化提升 W5-8（當前）| 🎯高峰 W9-13 | ⬇️減量 W14-16

[週曆]
  週一卡片 | 週三卡片 | 週五卡片 | 週日卡片 | 休息日卡片...

[週評估]（每週最後訓練日完成後出現）

[匯出列]
  📄 存 PDF | 📱 下載手機版 HTML
```

### 日卡片

- 類型 badge：輕鬆跑（綠）｜節奏跑（橘）｜間歇跑（紅）｜長跑（藍）｜休息（灰）｜減量週（橘邊）
- 今日高亮：特殊邊框 + "今天" 標記
- 已完成：綠色底 + 勾選打字覆蓋
- 點擊「標記完成」→ 記錄到 log

---

## 六、錯過訓練日

用戶點「我跳過了」：
1. 標記該日為 `missed`，計入遺漏次數
2. 系統檢查同週剩餘休息日（今天之後）
3. **有可用休息日**：彈出「要補跑嗎？把 [週X] 改為補跑日」→ 確認後休息日變訓練卡
4. **無可用休息日 / 已過週日**：直接記遺漏，不調整計畫
5. 補跑內容 = 原計畫課表（縮短 20%）

---

## 七、週評估（Check-in）

每週最後訓練日完成後，在計畫視圖底部出現評估卡片。

**5 個問題（勾選）**：
- 本週所有訓練都完成了
- 身體無異常疲勞或疼痛
- 睡眠品質良好
- 長跑結束後隔天恢復正常
- 節奏跑 / 間歇跑配速達標

**結果判定**：

| 勾選數 | 決定 | 下週調整 |
|--------|------|---------|
| 5 | 升級 | +10% 跑量，如達高峰期上限則維持 |
| 3–4 | 維持 | 維持現有計畫 |
| 1–2 | 減量 | 跑量 -15%，移除間歇 |
| 0 | 停止/就醫 | 顯示警告，建議休息一週後重評 |

## 八、訓練紀錄（Log View）

**輸入欄位**：日期、實際距離（km）、完成時間、體感強度 RPE（1–10）、備註

**統計卡片**：累積公里數、完成次數、計畫遵從率（%）

**紀錄表格**：日期｜類型｜距離｜時間｜配速｜RPE

## 九、匯出

### PDF
- `window.print()` 觸發
- `@media print` 隱藏：導覽列、工具列、匯出列、check-in 表單
- 列印版：進度卡 + 當週日卡（3 欄排版）

### 手機版 HTML
- JS Blob 下載，檔名 `trainer-week-{N}.html`
- 內含：當週 7 天卡片（含熱身/主課/收操）
- 勾選功能用 localStorage 保存（手機離線可用）
- 無外部依賴，單檔可獨立開啟

`site/index.html` 導覽列加入連結：`<a href="trainer.html">🗓 訓練計畫</a>`

## 十一、優先順序（分兩階段）

### Phase 1（核心生成器）
- 表單 9 欄位 + 即時計算
- 計畫生成邏輯（週曆 + 配速 + 階段）
- 計畫視圖（進度 + 週曆卡片 + 勾選）
- 減量週 + 賽前減量自動插入
- PDF 匯出
- HTML 匯出

### Phase 2（優化層）
- 週評估 + 自動調整
- 訓練紀錄 Log
- 錯過訓練日 → 補跑建議
- 休息日彈性訓練建議
- 肌力/核心課表附加
