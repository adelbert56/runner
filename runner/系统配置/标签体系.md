# 跑者广场 - 标签体系

清晰的标签体系帮助内容分类和检索。使用以下标签规范。

## 一级标签（内容类别）

### 📍 赛事 `#赛事`
所有与赛事相关的内容（赛事公告、赛事分析、参赛指南等）

### 💭 心得 `#心得`
个人分享（训练日志、比赛体验、装备评测、营养经验等）

### 📚 攻略 `#攻略`
教程、建议、技巧分享（不限于个人体验）

### 📊 分析 `#分析`
数据统计、趋势分析、排行榜等

---

## 二级标签（子分类）

### 赛事细分 `#赛事/...`
- `#赛事/马拉松` - 42.2km马拉松赛
- `#赛事/半马` - 21.1km半马赛
- `#赛事/路跑` - 其他路跑竞赛（5km-10km等）
- `#赛事/越野` - 越野跑、登山赛事
- `#赛事/铁人` - 三项铁人赛
- `#赛事/线上` - 线上虚拟赛

### 心得细分 `#心得/...`
- `#心得/装备` - 跑鞋、服装、手表、GPS等
- `#心得/训练` - 训练日志、课表设计、训练感受
- `#心得/比赛` - 赛事体验、成绩分享、赛后复盘
- `#心得/营养` - 饮食、补给、营养计划

### 地理位置 `#地点/...`
- `#地点/台中` - 台中市
- `#地点/南投` - 南投县
- `#地点/彰化` - 彰化县
- `#地点/苗栗` - 苗栗县
- `#地点/其他` - 其他地区

### 难度等级 `#难度/...`
- `#难度/初级` - 入门级（<21km或<2小时）
- `#难度/中级` - 中等（21-42km或2-4小时）
- `#难度/高级` - 挑战级（>42km或登山/越野）

### 其他实用标签
- `#推荐` - 编辑推荐内容
- `#热门` - 讨论热烈/点击高
- `#新手` - 适合跑步初心者
- `#装备测评` - 重点装备评测
- `#长期计划` - 长期系列文章

## 标签使用示例

### 比赛报告
```
比赛报告：2026台中都會馬拉松

标签: #赛事 #赛事/马拉松 #心得 #心得/比赛 #地点/台中 #难度/中级
```

### 装备评测
```
评测：Asics Novablast 3 跑鞋

标签: #心得 #心得/装备 #推荐 #新手
```

### 训练分享
```
12周马拉松训练计划

标签: #心得 #心得/训练 #攻略 #难度/中级
```

### 赛事分析
```
2026年5月中部赛事热度排行

标签: #分析 #赛事 #地点/台中 #热门
```

## 标签使用规则

📌 **必须标签** (每篇至少选1个):
- 一级标签：`#赛事` / `#心得` / `#攻略` / `#分析`

📌 **推荐标签** (2-3个):
- 二级标签（子分类）
- 地理位置
- 难度等级
- 其他实用标签

📌 **标签数量**:
- 最少: 2个
- 最多: 5个
- 避免过度标签化

## Obsidian中的标签操作

### 添加标签
```markdown
正文中直接写: #标签名
或用Obsidian Tag面板添加

例：
这是关于 #赛事/马拉松 的内容...
```

### 搜索标签
```
Obsidian中: Ctrl+Shift+F (全局搜索)
输入: tag:#赛事
或点击标签自动展开
```

### 标签图谱
```
点击Obsidian左侧Graph View查看标签关联
```

## FAQ

**Q: 一篇文章可以用多个一级标签吗?**  
A: 通常不建议。如需要，最多2个（如：#赛事 #攻略）。

**Q: 自定义标签可以吗?**  
A: 不建议。保持体系统一便于检索。特殊情况可向管理员提议新增标签。

**Q: 标签名大小写有要求吗?**  
A: 建议全小写，避免重复。
