# 跑者广场 - 標籤體系

清晰的標籤體系帮助內容分类和檢索。使用以下標籤规范。

## 一級標籤（內容类别）

### 📍 賽事 `#賽事`
所有与賽事相关的內容（賽事公告、賽事分析、参赛指南等）

### 💭 心得 `#心得`
個人分享（訓練日誌、比賽體驗、裝備評測、营养经验等）

### 📚 攻略 `#攻略`
教程、建議、技巧分享（不限于個人體驗）

### 📊 分析 `#分析`
数据统计、趋势分析、排行榜等

---

## 二级標籤（子分类）

### 賽事细分 `#賽事/...`
- `#賽事/馬拉松` - 42.2km馬拉松赛
- `#賽事/半马` - 21.1km半马赛
- `#賽事/路跑` - 其他路跑竞赛（5km-10km等）
- `#賽事/越野` - 越野跑、登山賽事
- `#賽事/鐵人` - 三项鐵人赛
- `#賽事/線上` - 線上虛擬赛

### 心得细分 `#心得/...`
- `#心得/裝備` - 跑鞋、服裝、手錶、GPS等
- `#心得/訓練` - 訓練日誌、课表设计、訓練感受
- `#心得/比賽` - 賽事體驗、成绩分享、賽後復盤
- `#心得/营养` - 饮食、補给、营养計畫

### 地理位置 `#地點/...`
- `#地點/台中` - 台中市
- `#地點/南投` - 南投县
- `#地點/彰化` - 彰化县
- `#地點/苗栗` - 苗栗县
- `#地點/其他` - 其他地区

### 難度等級 `#難度/...`
- `#難度/初級` - 入門級（<21km或<2小時）
- `#難度/中級` - 中等（21-42km或2-4小時）
- `#難度/高级` - 挑战级（>42km或登山/越野）

### 其他实用標籤
- `#推薦` - 編輯推薦內容
- `#熱門` - 讨论热烈/点击高
- `#新手` - 适合跑步初心者
- `#裝備测评` - 重点裝備評測
- `#长期計畫` - 长期系列文章

## 標籤使用示例

### 比賽报告
```
比賽报告：2026台中都會馬拉松

標籤: #賽事 #賽事/馬拉松 #心得 #心得/比賽 #地點/台中 #難度/中級
```

### 裝備評測
```
評測：Asics Novablast 3 跑鞋

標籤: #心得 #心得/裝備 #推薦 #新手
```

### 訓練分享
```
12周馬拉松訓練計畫

標籤: #心得 #心得/訓練 #攻略 #難度/中級
```

### 賽事分析
```
2026年5月中部賽事热度排行

標籤: #分析 #賽事 #地點/台中 #熱門
```

## 標籤使用規則

📌 **必须標籤** (每篇至少选1个):
- 一級標籤：`#賽事` / `#心得` / `#攻略` / `#分析`

📌 **推薦標籤** (2-3个):
- 二级標籤（子分类）
- 地理位置
- 難度等級
- 其他实用標籤

📌 **標籤数量**:
- 最少: 2个
- 最多: 5个
- 避免过度標籤化

## Obsidian中的標籤操作

### 添加標籤
```markdown
正文中直接写: #標籤名
或用Obsidian Tag面板添加

例：
这是关于 #賽事/馬拉松 的內容...
```

### 搜索標籤
```
Obsidian中: Ctrl+Shift+F (全局搜索)
输入: tag:#賽事
或点击標籤自動展开
```

### 標籤图谱
```
点击Obsidian左侧Graph View查看標籤关联
```

## FAQ

**Q: 一篇文章可以用多個一級標籤嗎?**
A: 通常不建議。如需要，最多2个（如：#賽事 #攻略）。

**Q: 自訂標籤可以嗎?**
A: 不建議。保持體系統一便於檢索。特殊情況可向管理員提議新增標籤。

**Q: 標籤名大小寫有要求嗎?**
A: 建議全小写，避免重复。
