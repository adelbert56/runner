# 🏃 歡迎來到跑者廣場 (Runner Plaza)

中部跑步社區知識庫 × 賽事聚合平台

---

## 🎯 這是什麼？

**跑者廣場**是一個以Obsidian為核心的跑步社區平台，目標是為台灣中部跑者提供：
- 📅 **賽事信息聚合** - 自動爬取+整理中部各大馬拉松、路跑、越野賽事
- 💭 **心得分享** - 裝備評測、訓練攻略、比賽報告
- 📊 **數據分析** - 賽事熱度、參賽趨勢、個人成績追蹤
- 🌐 **網站展示** - 未來將建立網站/App對外展示內容

## 🚀 快速開始

### 我想投稿
1. 閱讀 [[投稿指南]]
2. 準備內容（比賽報告/裝備評測/訓練分享）
3. 發送到 **投稿群組**（Line/Telegram）
4. 等待24小時內審核 ✓

### 我想查找賽事
1. 進入 [[赛事]] 文件夾
2. 按月份或難度篩選
3. 點擊報名鏈接或查看攻略

### 我想學習跑步知識
1. 進入 [[心得]] 文件夾
2. 用標籤篩選（如：#新手 #装备 #训练）
3. 閱讀相關文章和推薦
